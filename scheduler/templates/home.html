Introduction

Welcome to the Hadar Server Demo. Hadar is an open-source adequacy python library. This website is the demo server. Hadar can compute study locally but can also send to a server. Before using this server, we strongly advertise you to train with Hadar step by step.

You can find more information on Hadar and examples at hadar-simulator.org, code source is hosted on github.

Send compute on this server

Switch from locally compute to cloud compute is very simple. Inside you Hadar code, you should have a line like: 

optim = hd.LPOptimizer()

You just need to change by: {% if token %} 
optim = hd.RemoteOptimizer(url='https://{{ host }}', token='****')

{% else %} 
optim = hd.RemoteOptimizer(url='https://{{ host }}')

{% endif %}

Deploy your own server

Server source code is available on github, it's packaged in docker image too. In the repository, you have a ready to use docker-compose config to deploy your server. A copy is printed below:
